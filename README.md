DJF
========

有关使用说明，请查看 [WIKI](https://github.com/changyou/webstart/wiki)

## 使用步骤

1. 下载代码

  $> git clone https://github.com/fengliu222/djf-active-lottery.git

2. 安装FED:

  $> npm install -g fed

3. 启动：

  $> cd djf-active-lottery
  $> fed server -w ./devConfig.json

4. 访问：

  http://localhost:3000

5. 发布前执行：

  grunt build
